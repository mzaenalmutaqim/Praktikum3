# Program Menghitung Luas dan Keliling Lingkaran dengan Bahasa Pemograman PYTHON

Rumusnya adalah sebagai berikut:

    r = input("Masukkan jari-jari lingkaran : ")
    pi = 3.14
    l = pi * int(r) * int(r)
    k = 2 * pi * int(r)
    print("Luas Lingkaran : ",l)
    print("Keliling Lingkaran : ",k)
